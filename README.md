# Medical Timeline Chart

A modern, interactive medical timeline chart designed to visually display the sequence of actions taken by medical personnel. This tool is especially useful for visualizing complex medical cases, such as those involved in medical malpractice lawsuits, but can also be adapted for general clinical documentation and analysis.

## Features

- **Interactive Charts:** Built with Chart.js for dynamic visualization of vital signs, interventions, transfusions, and medical orders.
- **Tabbed Interface:** Easily switch between different care phases (PACU/FCC, Operation Room/ICU) for clear separation of events.
- **Time Range Selection:** Flexible time selectors allow users to focus on specific periods of interest.
- **Responsive Design:** Optimized for both desktop and mobile devices using Bootstrap 5.
- **Data Anonymization:** All patient and provider names are anonymized for privacy and portfolio demonstration.
- **Customizable:** Easily adapt the chart to new data or additional medical events.
- **Export/Print:** Print-ready chart views for documentation or sharing.

## Usage

Open `index.html` in your browser. No backend or build step required — all logic is client-side JavaScript.

## Technologies Used

- HTML5, CSS3, JavaScript (ES6)
- [Chart.js](https://www.chartjs.org/)
- [Bootstrap 5](https://getbootstrap.com/)
- [GLightbox](https://biati-digital.github.io/glightbox/)

## Example Use Cases

- Visualizing the timeline of care in a medical malpractice case
- Presenting clinical events for case reviews or audits
- Educational demonstrations for medical students
